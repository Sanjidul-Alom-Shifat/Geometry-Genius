The full meaning of DOM is (Document Object Modal). Dom is a programming interface for web documents. By using DOM we can grave the html elements and make them dynamic. DOM is the way to connect the html file to a Javascript file.

There are many ways to select elements from a DOM :

1. getElementById : This method search a element by element id.

2. getElementsByClassName : This method search a element by class name.

3. getElementsByTagName : This method search a element by tag name. Example : like span , div

4. querySelector : This method use to returns the first element that matches the specified selector .

5. querySelectorAll : This method use to returns elements that match the specified selector .

Event Bubbling is a concept in the DOM (Document Object Model). Event bubbling is the propagation of an event that stops from its origin up to the root element.

When a function is sent as an argument to a function, it is executed after the main function is executed, so it is called a callback function. By using a callback function you can wait for the result of the previous function call and then execute another function call.
